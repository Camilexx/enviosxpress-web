# ✍️ Copywriting Pro Skill

> Write copy that sells. Every word has a job — if it's not converting, it's wasting space.

---

## Quick Reference

```
THE 4 COPY FORMULAS YOU NEED:
1. HERO:     PAS (Problem → Agitate → Solution)
2. FEATURES: FAB (Feature → Advantage → Benefit)
3. PRICING:  VALUE-FIRST (value before price)
4. CTA:      VERB + BENEFIT + NO FRICTION
```

---

## Formula 1: Headlines (PAS)

### Structure
```
[Resultado Deseado] + [en Tiempo Específico] + [Sin/Con Eliminación de Objeción]
```

### Examples by Industry

| Industry | Headline |
|----------|----------|
| **Courier** | "Envía paquetes a todo Ecuador desde $2.50 — entrega en 24h" |
| **SaaS** | "Duplica tus ventas en 30 días sin contratar más vendedores" |
| **E-commerce** | "Tu tienda online lista en 48 horas — sin código, sin complicaciones" |
| **Fintech** | "Cobra a tus clientes al instante desde cualquier dispositivo" |
| **Education** | "Domina JavaScript en 12 semanas con proyectos reales" |

### Subheadlines (expand the benefit)
```
FORMULA: Explicar CÓMO se logra el resultado del headline

"Nuestra red de 200+ puntos de entrega y tecnología de 
seguimiento en tiempo real garantiza que cada paquete 
llegue a su destino, a tiempo, siempre."
```

## Formula 2: Feature Bullets (FAB)

### Structure
```
[Verbo de acción] + [qué hace concretamente] + [por qué importa al usuario]
```

### Template
```
✓ [Feature]
  [Advantage] →→ [Benefit for user]

EXAMPLES:
✓ Seguimiento GPS en tiempo real
  Ve exactamente dónde está tu paquete →→ Nunca más llames preguntando "¿dónde está?"

✓ Cobro contra entrega
  Tu cliente paga al recibir el paquete →→ Aumenta tus ventas eliminando el miedo a comprar online

✓ Dashboard de analytics
  Métricas de envíos en un solo lugar →→ Toma decisiones basadas en datos, no en intuición
```

## Formula 3: Value Propositions (3 Pillars)

### The Rule of Three
```
Siempre presenta exactamente 3 propuestas de valor.
El cerebro humano procesa y recuerda mejor en grupos de 3.

STRUCTURE PER PILLAR:
├── Emoji/Icon (visual anchor)
├── Title (2-4 words max)
├── Description (1-2 sentences)
└── Proof point (metric or quote)
```

### Example Set
```
⚡ Rapidez Garantizada
   Entrega en 24-48 horas a nivel nacional.
   "98.5% de paquetes entregados a tiempo"

🛡️ Seguro y Rastreable
   Seguimiento GPS en tiempo real + seguro incluido.
   "0 paquetes perdidos en los últimos 6 meses"

💰 Precio Justo
   Tarifas desde $2.50 sin costos ocultos.
   "40% más económico que la competencia"
```

## Formula 4: CTA Copy

### Primary CTA (max conversion)
```
FORMULA: [Verbo de Acción] + [Beneficio Inmediato]

✅ GOOD CTAs:
   "Cotizar envío gratis"
   "Empezar prueba de 14 días"
   "Crear cuenta en 30 segundos"  
   "Descargar guía gratuita"
   "Reservar mi lugar"
   "Ver demostración"

❌ BAD CTAs:
   "Enviar" (genérico, sin beneficio)
   "Registrarse" (suena a compromiso)
   "Click aquí" (no dice qué pasa)
   "Comprar ahora" (demasiado agresivo en landing)
   "Suscribirse" (implica pago recurrente)
```

### Secondary CTA (for hesitant visitors)
```
"¿No estás listo? Ve cómo funciona primero →"
"Habla con un asesor sin compromiso"
"Descarga nuestro caso de éxito"
```

## UI Microcopy

### Form Labels & Placeholders
```
RULE: Labels describe the field. Placeholders show format.

✅ Label: "Email de trabajo"  |  Placeholder: "tu@empresa.com"
✅ Label: "Contraseña"        |  Placeholder: "Mínimo 8 caracteres"  
✅ Label: "Teléfono"          |  Placeholder: "+593 9XX XXX XXXX"

❌ Label: "Email"             |  Placeholder: "Ingrese su email" (redundant)
```

### Error Messages
```
FORMULA: [Qué salió mal] + [Cómo arreglarlo]

✅ "El email no es válido. Verifica que tenga formato correcto (ej: tu@email.com)"
✅ "La contraseña es muy corta. Necesitas al menos 8 caracteres"
✅ "Este email ya tiene cuenta. ¿Quieres iniciar sesión?"

❌ "Error 422: Validation failed" (técnico, no humano)
❌ "Campo obligatorio" (no dice qué hacer)
```

### Success Messages
```
✅ "¡Cuenta creada! Revisa tu email para confirmar"
✅ "Pago procesado exitosamente. Recibirás confirmación por email"
✅ "Cambios guardados"

❌ "Operación exitosa" (genérico)
❌ "Success" (en inglés si el usuario es hispanohablante)
```

### Empty States
```
FORMULA: [Qué está vacío] + [Por qué importa] + [CTA para poblar]

✅ "Aún no tienes envíos
    Cuando crees tu primer envío, aparecerá aquí con seguimiento en tiempo real.
    [Crear primer envío →]"

❌ "No hay datos" (frío, sin dirección)
```

### Loading States
```
✅ "Calculando tarifa..."
✅ "Procesando pago..."
✅ "Preparando tu dashboard..."

❌ "Loading..." (genérico)
❌ "Please wait..." (en inglés)
```

## SEO Meta Tags

### Formula
```
TITLE TAG (50-60 chars):
[Keyword Principal] — [Beneficio] | [Marca]
"Envíos a Todo Ecuador — Entrega en 24h | EnviosXpress"

META DESCRIPTION (150-160 chars):
[Qué haces] + [Beneficio principal] + [CTA implícito]
"Servicio de courier a nivel nacional desde $2.50. Entrega en 24-48h 
con seguimiento en tiempo real. Cotiza gratis en 30 segundos."

OG TITLE (shorter, punchier):
"Envíos a todo Ecuador desde $2.50 🚀"

OG DESCRIPTION:
"El courier más rápido y confiable. +15,000 envíos entregados."
```

### Per Page Template

| Page | Title | Description |
|------|-------|-------------|
| Home | `[Marca] — [Propuesta de valor principal]` | `[Qué haces] + [beneficio] + [CTA]` |
| Pricing | `Precios — Planes desde $X/mes \| [Marca]` | `Planes flexibles para [audiencia]. Compara features y elige.` |
| About | `Sobre Nosotros — [Marca]` | `[Historia] + [misión] + [diferenciador]` |
| Blog Post | `[Título del Post] — [Marca]` | `[Resumen en 150 chars] que genere curiosidad` |

## Email Copy

### Welcome Email
```
Subject: ¡Bienvenido a [Marca]! Tu primera misión 🎯
Preview: Tu cuenta está lista. Esto es lo primero que debes hacer.

Body:
Hola [Nombre],

¡Bienvenido a [Marca]! 🎉

Tu cuenta está lista. Aquí tienes tu primera misión:

→ [CTA principal: lo más importante que deben hacer]

Si necesitas ayuda, responde a este email. 
Un humano real te contestará en menos de 2 horas.

— El equipo de [Marca]
```

### Payment Confirmation
```
Subject: Pago confirmado — Recibo #[número]
Preview: Tu pago de $[monto] fue procesado exitosamente.

Body:
Hola [Nombre],

Tu pago ha sido procesado ✅

Detalle:
• Plan: [Nombre del plan]
• Monto: $[monto]
• Próximo cobro: [fecha]

[Ver factura completa →]

¿Preguntas? Respondemos en < 2 horas.
```

## Tone of Voice Guidelines

### The Brand Voice Spectrum
```
CASUAL ←──────────────────────────────→ FORMAL

For most SaaS/startups, aim here: ████████░░
(Professional but approachable, never corporate-stiff)

DO:
✅ Use "tú" not "usted" (unless enterprise B2B)
✅ Use contractions ("no es" → "no's" sparingly)
✅ Write like explaining to a smart friend
✅ Use specific numbers ("98.5%") not vague ("muchos")
✅ Active voice ("Enviamos tu paquete" not "Tu paquete será enviado")

DON'T:
❌ Use jargon without explaining
❌ Use "soluciones innovadoras", "sinergias", "ecosistema disruptivo"
❌ Be overly casual (no memes in error messages)
❌ Mix languages ("Haz click en submit")
```
